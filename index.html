How to Play

Codewords are like crosswords, but with no clues. Every letter of the alphabet has been replaced by a number (1-26). The same number always represents the same letter.

A few letters are revealed to get you started. Use these to deduce other letters and complete the grid with valid English words.

Controls: Click a cell, then type a letter. Use arrow keys to navigate. Press Delete/Backspace to clear a cell.
